EXAMPLES OF EXTERNALS IN C LANGUAGE FOR PURE DATA. 
This is a repository with pedagogical purposes only.
The authors of each code, bibliography or resource used are cited as fair as possible. If you find 
something which you feel you would claim, just let me know.
In order to build the externals the user must have installet cmake version 4.3.3 and the proper C compiler for each system.
There are prebuild externals for Windows 11, but it is strongly advisable to rebuild.
Both the folders ./control and ./audio contain the sources and CMakeKists to build. The same CMakeKists should work in Windows, 
Linux and Mac OS.
Both the folders ./control and ./audio also contain files named "commands.txt" with the basic cmake commands to build.
