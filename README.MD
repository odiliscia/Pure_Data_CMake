# EXAMPLES OF EXTERNALS IN C LANGUAGE FOR PURE DATA

This is a repository with pedagogical purposes only. The authors of each code, bibliography, or resource used are cited as fairly as possible. If you find something which you feel you should claim, just let me know.

## Prerequisites

In order to build the externals, the user must have installed:
* **CMake** (version 4.3.3 or higher)
* A proper **C compiler** for your specific operating system.

> 💡 **Note:** There are prebuilt externals for Windows 11 included, but it is strongly advisable to rebuild them.

## How to Build

Both the `./control` and `./audio` folders contain the sources and `CMakeLists.txt` to build the project. The same `CMakeLists.txt` should work across **Windows, Linux, and macOS**.

Inside both folders, you will also find a file named `commands.txt` containing the basic CMake commands required for compilation.
